¿Cuál es la diferencia entre Internet y la World Wide Web?

Internet es el medio por el cual funciona y www son las aplicaciones y todo lo demás.

¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

  • Get: hace un request de información, sin modificarla.
  • Head: se emplea para solicitar información de lo que esta realizando la solicitud sin transferirlo como tal, se parece al get.
  • Post: Se usa para publicar información al servidor
  • Patch: Actualiza información
  • Delete:Actualiza información

¿Qué método HTTP se debe utilizar al enviar un formulario HTML, por ejemplo cuando ingresas tu usuario y contraseña en algún sitio? ¿Por qué?

Post, debido que está publicando información.

¿Qué método HTTP se utiliza cuando a través de un navegador web se accede a una página a través de un URL?

Get, ya que pide información correspondiente al Url de un servidor.

Un servidor web devuelve una respuesta HTTP con código 200. ¿Qué significa esto? ¿Ocurrió algún error?

No es un error. Significa que la página a cargado correctamente.

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 404? ¿Por qué?

Significa que la página a la que quieres acceder, no se encuentra disponible.

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 500? ¿Por qué?

Es un error que implica un error interno en el servidor.

¿Cuáles son las diferencias principales entre HTML 4 y HTML5?

HTML5 tiene un mejor manejo de atributos y tags, y tiene un mejor manejo de errores.

¿Qué componentes de estructura y estilo tiene una tabla?

Puede usar tres: table, tr que maneja rows y td para manejo de celdas.

¿Cuáles son los principales controles de una forma HTML5?

  • Number: crea un campo de entrada para el ingreso de un dato numérico
  • Date: Crea un campo que permite la seleccion de fecha/hora
  • Color: Crea un campo donde se puede escoger dentro de una rama de colores
  • Search: Crea un campo de busqueda

Sobre el ciclo de vida y desarrollo de los sistemas de información:
¿Cuál es el ciclo de vida de los sistemas de información?

Planificación, análisis, diseño, implementación, pruebas, instalación uso y mantenimiento.

¿Cuál es el ciclo de desarrollo de sistemas de información?

Implementación, pruebas, despliegue, uso y mantenimiento.

Como ingeniero de software ¿cuál es tu recomendación sobre el uso de !important en un CSS?

Mejora la visualización de la página volviendola más user friendly, facilitando el manejo para el usuario

Si se pone una imagen de fondo en una página HTML, ¿por qué debe escogerse con cuidado?

Se debe tomar en cuenta el tamaño de la imagen y observar que los colores de la imagen no perjudiquen al texto de la página

Como ingeniero de software, ¿cuál es tu recomendación al elegir las unidades de un propiedad de estilo entre %, px y pt?

Al usar px, el usuario no puede modificar el tamaño de la letra en caso de que requiera hacerlo más grande con IE

¿Por qué el uso de una versión minimizada del CSS mejora el rendimiento del sitio?

Mejora el procesamiento de la página, y en cuestión de desarrollo facilita para futuras modificaciones.
