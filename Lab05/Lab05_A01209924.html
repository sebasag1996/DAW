Cuestionario 6 (Font change)

  • static: los elementos aparecen en el orden del documento.
  • absolute: el elemento es posicionado respecto a su primer ancestro no estático (static).
  • fixed: el elemento es posicionado respecto a la ventana.
  • relative: el elemento es posicionado respecto a su posición normal.
  • sticky: el elemento es posicionado respecto a la posición del scroll del usuario.
  • initial: configura a default (visible).
  • inherit: heredar propiedad del elemento padre.

¿Cuáles son los valores estándar para la propiedad visibility?

  • visible: el elemento es visible (Valor Estándar).
  • hidden: el elemento es invisible y ocupa espacio.
  • collapse: para tablas. Esconde filas o columnas sin afectar la tabla.
  • initial: configura a default (visible).
  • inherit: heredar propiedad del elemento padre.
