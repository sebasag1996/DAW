Cuestionario de Javascript

¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Ambos lenguajes manejan lo que es programación orientada objetos, y pueden ser corridos en un servidor web. Por otra parte, en cuanto a diferencias, JavaScript es un lenguaje para realizar scripts orientados a objetos. Java puede ser corrido en un compilador o en un navegador, JavaScript solo puede ser corrido dentro de un navegador, y principalmente sirve para hacer las paginas responsivas e interactivas.

¿Qué métodos tiene el objeto Date?

  • getDate()

    Devuelve el día del mes
  • getDay()

    Devuelve el día de la semana
  • getFullYear()

    Devuelve el año en números
  • getHours()

    Devuelve la hora en formato 24Hrs
  • getMilliseconds()

    Devuelve los milisegundos en un rango de 0 a 999.

    • ¿Qué métodos tienen los arreglos?

  • toString()

    Devuelve un string con todos los datos del arreglo
  • pop()

    Elimina el último item del arreglo
  • push()

    Agrega un elemento al arreglo al último
  • shift()

    Elimina el primer elemneto del arreglo y desplaza los demás
  • unshift()

    Agrega un item al inicio y recorre los demás

    • ¿Cómo se declara una variable con alcance local dentro de una función?

    La variable se vuelve local al estar dentro del la función, con el parámetro let.

    ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

    El valor de la variable después de modificarla se debe mantener para otras funciones del programa.

    ¿Qué método de String se puede utilizar para buscar patrones con expresiones regulares?¿Para qué podrías utilizar esto en una aplicación web?

    El método se llama search, este busca expresiones regulares dentro de un String, podría tener aplicación en aplicaciones que se basen en buscadores, catálogos, etc.
