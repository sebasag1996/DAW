¿Por qué es importante hacer un session_unset() y luego un session_destroy()?

Para poder restablecer las variables de la sesión y para poder terminar la sesión y hacer un buen uso de memoria y dejar de guardar las variables cuando no se utilizan.

¿Cuál es la diferencia entre una variable de sesión y una cookie?

Una variable de sesión se guarda en el servidor y la cookie se guarda localmente.

¿Qué técnicas se utilizan en sitios como facebook para que el usuario no sobreescriba sus fotos en el sistema de archivos cuando sube una foto con el mismo nombre?

Les cambia el nombre a los archivos para que 2 archivos no tengan la misma ruta, es decir le puede agregar caracteres o le pide al usuario que lo nombre de otra manera.

¿Qué es CSRF y cómo puede prevenirse?

El CSRF es un tipo de exploit malicioso de un sitio web en el que comandos no autorizados son transmitidos por un usuario en el cual el sitio web confía, Se puede prevenir validando cada input con htmlspecialchars
